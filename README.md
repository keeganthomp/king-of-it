## King of It is an application that gives users a daily challenge and rewards users for completing said daily challenge.

- There will be a winner at the end of each month
- To Complete a challenge, a user must upload a photo or video of them completing a challenge
- Photos will be examined by moderators to ensure they do entail a user completing the challenge
- There will be a stream of photos related to each challenge so users can view others completing the challenge

### King of It is built using MongoDB, Express, React, & NodeJS (MERN).
